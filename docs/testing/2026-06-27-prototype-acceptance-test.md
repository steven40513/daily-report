## 修復紀錄（2026-06-27）

### Bug 1：人員總數不即時更新（T2-02~T2-05, T3-01~T3-02）

- **原因：** `adjustCrew()` 只更新單行數字，未重新加總底部「今日總人數」
- **修復：** 新增 `updateCrewTotal()` 函數，在 `adjustCrew()` 末尾呼叫，遍歷所有 `.crew-num` 加總後寫入底部顯示區
- **驗證：** 6 個相關測試全部通過

### Bug 2：innerHTML XSS 注入風險（T2-14）

- **原因：** `confirmAddItem()` 使用 `innerHTML` 拼接使用者輸入的名稱，`<script>` 或 `<img onerror>` 等標籤會被瀏覽器解析
- **修復：** 改用 `document.createElement()` + `textContent` 逐一建立 DOM 元素，使用者輸入永遠以純文字呈現
- **驗證：** 3 個 XSS 測試場景（script、img onerror、特殊字元）全部通過
